#include <muu/quaternion.h>
template<typename Scalar>
muu::euler_rotation struct

A set of euler angles used for rotation.

Template parameters
Scalar The scalar component type of the data members.

Contents

  • Reference
    • Public variables

Strictly speaking this type models a specific form of Euler angles relating to the Aircraft Principal Axes, and by that convention the rotations contained within are always applied in the member order: Yaw, then Pitch, then Roll.
