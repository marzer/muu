template<typename Scalar, size_t Dimensions>
static delta_type muu::vector<Scalar, Dimensions>::angle(const vector& v1, const vector& v2) constexpr noexcept

Calculates the angle between two vectors.

Parameters
v1 A vector.
v2 A vector.
Returns The angle between v1 and v2 (in radians).

Remark

The angle returned is the unsigned angle between the two vectors; the smaller of the two possible angles between the two vectors is used. The result is never greater than pi radians (180 degrees).

Conditional availability

This function is only available when dimensions == 2 or 3.
