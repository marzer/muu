template<impl::fixed_string_udl_impl Str>
auto muu::literals::operator""_fs() consteval

Constructs a fixed_string directly using a string literal.

constexpr auto str = "hello"_fs;
static_assert(str == fixed_string{ "hello" })

Conditional availability

This operator depends on C++20's class-type NTTPs and three-way comparison operator. Check for support using MUU_HAS_FIXED_STRING_LITERALS and/or build::supports_fixed_string_literals.
