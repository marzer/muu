void* muu::generic_allocator::allocate(size_t size)

Requests a memory allocation.

Parameters
size The size of the requested allocation.
Returns When exceptions are enabled: A pointer to the new allocation, or throws std::bad_alloc.
When exceptions are disabled: A pointer to the new allocation, or nullptr.

Remark

Allocations returned by this overload will have an alignment of default_alignment.
