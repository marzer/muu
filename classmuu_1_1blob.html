muu::blob class
#include <muu/blob.h>

Interface for managing chunks of memory.

Contents

  • Reference
    • Constructors, destructors, conversion operators
    • Public functions

A blob is effectively an RAII wrapper around muu::aligned_alloc, muu::aligned_realloc and muu::aligned_free, intended to be used anywhere you might previously have used something like std::vector<std::byte>.
