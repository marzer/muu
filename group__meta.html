Metafunctions and type traits module

Metaprogramming utilities to complement those found in <type_traits>.

Contents

  • Reference
    • Typedefs
    • Variables

Remark

Many of these are mirrors of (or supplementary to) traits found in the standard library's <type_traits>, but with simpler/saner default behaviour (e.g. most metafunctions treat references to types as if they were the actual type, because that typically makes more intuitive sense).
