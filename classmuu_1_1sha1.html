sha1& muu::sha1::finish() noexcept

Finishes calculating the hash.

Appending to the hash function's input has no effect after finish() is called.

const hash_type& muu::sha1::value() const noexcept

Returns the calculated hash value.

Warning

Calling this before finish() has been called is undefined behaviour.
